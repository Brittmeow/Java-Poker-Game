# Java-Poker-Game
WORK IN PROGRESS
Currently under construction/next up to work on: additional rounds 
Future additions: "discard pile" to shuffle and add back to round deck, betting

Current functionality:
Deals the player and computer cards, sorts the hands, swaps computer hand, asks to swap cards (up to 3), sorts hands again, evaluates the player and computer hands, and compares the player and computer hands to find the round winner
